Chores Quest
App Overview

ChoresQuest.com is a web-based family task management system designed to help parents assign, track, and reward children's chores. The app supports two user roles — Parent and Child — with separate dashboards tailored to their needs. Parents can create tasks with repeat schedules, assign them to one or more children, attach icons or images, and set reward points. Children log in through a simplified interface to view their tasks, mark them complete, and redeem points in a gamified and engaging environment.

Landing Page
Users choose to log in as a parent or child, with options to sign up, log in, or install the app as a home screen shortcut.
Sign-Up Screen
Parents create a secure account using name, username, and password, and receive a one-time recovery code for account recovery.
Login Screen
Parents log in with their credentials or use the recovery code if they forget their username or password.
Terms of Use
Outlines parent responsibilities, proper use, PIN protection, and the importance of storing the recovery code securely.
Privacy Policy
Emphasizes data safety, no use of real names or photos for children, and the secure handling of personal data.
Parent Dashboard Overview
Displays recent activity, active/overdue tasks, and navigation to manage notifications, tasks, rewards, and children.
Notifications Page
Shows a list of recent activities from family members and allows parents to mark notifications as read.
Manage Tasks
Parents can view, create, edit, duplicate, and delete tasks with details such as due dates, points, and assignments.
Task Editor
Customize task title, image/icon, notes, due date, repetition, and assign it to specific children or all children.
Manage Rewards
Add, edit, or delete rewards with images and point values that children can redeem from their dashboard.
Manage Children
Add or remove children, upload profile pictures, track points, and adjust scores manually if needed.
Settings
Configure PIN protection, enable notifications, auto-delete settings, and sync system time.
Child Profile Selection
Children tap their avatar to log in and access their personalized tasks and rewards.
Child Dashboard
Shows points, personal and family tasks, completed tasks, and a button to view available rewards.
Rewards View
Children can see rewards they can afford with their points and those they can’t yet claim.
